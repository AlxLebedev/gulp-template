Gulp 4

Проект собран с использованием сборщика Gulp 4. Настоящий проект может служить шаблоном для сборки верстки при помощи Gulp и в дальнейшем будет развиваться в плане подклчючения новых возможностей и функций.

Технологии в проекте

  • Gulp

    Gulp - это таск-раннер, инструмент, который позволяет автоматизировать любой ручной труд в любой области.

  • Sass/Scss

    Стили проекта написаны при помощи препроцессора SASS с использованием синтаксиса SCSS. В проекте используются custom properties для указания цветов.

  • CSS Grid

    Сетка выстроена с использованием технологии CSS Grid Layout

  • Pure JavaScript

    В проекте используется чистый JavaScript. Файлы "склеиваются" при помощи gulp-concat
